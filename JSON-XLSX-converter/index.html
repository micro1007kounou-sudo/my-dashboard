Excel ⇄ JSON 変換ツール

※ JSON は Excel に変換する際、表形式（2 次元データ）に変換されます。
　オブジェクトやネスト構造は自動的に展開されず、配列形式のデータのみ正しく扱えます。
※ JSON の階層構造（ネスト）、オブジェクト、キー名などは Excel 変換時に保持されない場合があります。
※ Excel は 1 シートのみ JSON に変換されます。複数シートがある場合、最初のシートのみが対象となります。
※ Excel 特有の情報（結合セル・書式・画像・コメントなど）は JSON 変換時にすべて失われます。
